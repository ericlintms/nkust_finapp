# Week 5: 課程內容 (Course Content)

## 學習目標

經過前幾週的努力，我們已經能把網路上的各家公司財報拉回來，乖乖存在資料庫裡。但光是知道本益比 (PE) 或是股價，沒什麼用呀！我們總不可能一檔一檔肉眼挑出便宜又賺錢的股票。

這週我們要請出 Python 資料分析界的最強神兵利器——**Pandas**！讓它幫我們一口氣處理成千上萬筆資料，幫每檔股票貼上「便宜」、「合理」或「昂貴」的標籤，實現我們「護城河價值投資儀」的最核心大腦。

## 涵蓋主題

1. **認識 Pandas 與 DataFrame**
   - 為什麼不用 List 跟 For 迴圈就算了？Pandas 到底在快什麼？
   - 什麼是 DataFrame (資料框)？（想像成在 Python 裡面跑超級加速版的 Excel 表格）。
   - Pandas 基本操作：讀取、篩選、新增欄位。

2. **實作核心財務邏輯 (三率分析與估值位階)**
   - **三率分析**：粗估毛利率、營業利益率、淨利率的關聯，判斷這是不是一門好生意。
   - **估值位階**：利用歷史本益比區間，算出現在的股價算是偏高還是偏低。
   - 如何在 DataFrame 裡，一行程式碼就把所有股票的「便宜/昂貴」標籤貼好。

3. **結合資料庫 (DB -> Pandas -> DB)**
   - 把 Week 3 的 SQLite 資料讀出來變成 Pandas DataFrame。
   - 用 Pandas 計算加工完畢後，再寫回資料庫。

## 本週預期產出

- 透過 Python Pandas 成功把我們的多檔股票數據轉換成一個結構化的表格 (DataFrame) 並印出來。
- 寫出一個運算邏輯，讓程式自動判斷一檔股票是不是符合「高 ROE 大於 15% 且本益比小於 15」的嚴格標準。
