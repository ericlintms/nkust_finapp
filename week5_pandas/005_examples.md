## 我們的練習清單

### 1. 例題一：Pandas 初體驗 (建立與篩選資料框)
- **對應檔案**: `examples/w501_pandas_basics.py`
- **練習目的**: 從無到有建立一個小小的股票資料框，接著體驗瞬間條件篩選的快感。
- **實作任務**:
  1. 切換目錄到 `week5_pandas/examples/`。
  2. 確保你安裝了 Pandas：在終端機輸入 `pip install pandas pyarrow` (pyarrow 可以讓 Pandas 跑更順)。
  3. 執行程式：`python w501_pandas_basics.py`。
  4. 觀察終端機，它會依序印出：原始表格、只留下大於市值 1 兆元的表格、新增了本益比欄位的表格。
  5. 試著打開程式碼，發揮想像力，請 AI 教你怎麼加一個「找出 P/E 小於 20 且市值最大的前兩名」的語法。

### 2. 例題二：選股神器的核心演算法實裝
- **對應檔案**: `examples/w502_calculate_indicators.py`
- **練習目的**: 實作我們剛才在理論課講到的「標籤化過濾邏輯」(被低估價值股 vs 昂貴股)。也就是體驗 Numpy + Pandas 混血特工的運算魅力。
- **實作任務**:
  1. 可能會需要安裝 numpy：`pip install numpy`。
  2. 執行程式：`python w502_calculate_indicators.py`。
  3. 注意看終端機印出來的最後那個 DataFrame。看看 AI 幫我們加上的 `value_label` 欄位有沒有乖乖按照我們的「高 ROE 配上 低 PE」的邏輯，精準判斷出哪些股票是好貨，哪些是妖股？
