# Week 1: 課程內容 (Course Content)

## 學習目標
這週是我們接觸 Vibe Coding (AI 輔助開發) 與 Python 財務邏輯封裝的起點。主要的目標是幫自己把接下來十八週開發會用到的基礎設施都先建立起來。

## 涵蓋主題
1. **開發環境建置**
   - 弄懂 Python 版本並建立虛擬環境 (Virtual Environment)。
   - 安裝 VS Code 與設定實用的擴充套件。
   - 整合 AI 輔助工具（本課程以 **Antigravity** 作為示範工具；若已有 GitHub Copilot 授權也可使用，兩者概念相通）。

2. **基礎 Python 複習**
   - 熟悉變數跟資料型態 (尤其是 Lists 和 Dictionaries)。
   - 掌握流程控制與函式 (Functions) 的寫法。

3. **財務邏輯實作**
   - 深入理解貨幣的時間價值 (Time Value of Money)。
   - 練習怎麼把現金流的計算邏輯轉化為 Python 程式碼。

## 本週預期產出
- 我能夠在自己的電腦上成功執行第一支 Python 程式。
- 完成一個能自動幫我算出淨現值 (NPV) 與內部報酬率 (IRR) 的 Python 腳本。
