# 實務範例與實作 (Week 2)

這週的練習，我們要一起把冷冰冰的 Python 變成可以在網頁上直接互動測試的服務。請到 `examples/` 資料夾底下打開這些程式碼來玩玩看。

## 我們的練習清單

### 1. 例題一：Hello World! 第一次發射 API

- **對應檔案**: `examples/w201_hello_api.py`
- **練習目的**: 確認我們電腦裡的 `fastapi` 和 `uvicorn` 工具都順利安裝好了，並玩玩看 `@app.get()` 這個把函數變網址的黑魔法。
- **實作任務**:
  1. **先切換到正確的工作目錄**：在終端機執行 `cd examples/`（從`week2_fastapi/` 資料夾進入）。uvicorn 需要在同一層目錄找到檔案才能成功啟動。
  2. 在終端機敲入這行啟動伺服器: `uvicorn w201_hello_api:app --reload`。
  3. 打開 Chrome 瀏覽器，前往這個有點酷的網址 `http://127.0.0.1:8000/docs`。
  4. 試著點擊 "Try it out"，執行 `/hello/{name}` 這支 API (敲進你自己的名字)，觀察伺服器吐給你的 JSON 格式長什麼樣子！

### 2. 例題二：把我們一週前寫的財務模型 API 化

- **對應檔案**: `examples/w202_npv_api.py`
- **練習目的**: 把我們 Week 1 寫好的 NPV 計算器，搬進 FastAPI 的大腦裡面。
- **操作技巧**: 這邊有一個很實用的技巧，就是要學會怎麼把「前端丟過來的一串文字」 (`'-10000, 3000, 4200, 6800'`)，在後端用 `.split(',')` 像切菜一樣切成一個個的項目，最後再把它們轉成 Python 的浮點數 List 才能做數學運算。
- **實作任務**: 來搞破壞吧！試著故意在我們剛剛看到的 Swagger UI 網頁中，傳入一個根本不能變成數字的奇怪字串 (例如傳個英文字母 'A')。觀察一下，我們的伺服器是會直接冒煙當機，還是會很優雅地回傳給我們一個設計好的 `400 Bad Request` 錯誤警告？
