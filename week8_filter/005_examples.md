# 實務範例與實作 (Week 8)

這週只有一場實戰！我們要用一支 Python 跟一個網頁檔，把我們 Week 1 ~ Week 8 所學的所有絕招全部濃縮在一起展現！這也是我們「護城河價值投資儀」這個 Web 版小專案的最核心靈魂。

```
week8_filter/
├── examples/
│   ├── templates/          
│   │   └── w801_screener.html  <-- 畫面上方有表單，下方有結果表格

```

## 我們的練習清單

### 🌟 例題：全自動智能選股器實裝
- **對應檔案**: `examples/w801_stock_screener.py` 與 `examples/templates/w801_screener.html`
- **練習目的**: 當使用者在網頁按下搜尋按鈕，親眼看見參數如何傳進 Python、被 Pandas 過濾並重新渲染成全新的結果。
- **實作任務**:
  1. 切換目錄到 `week8_filter/examples/`。
  2. 確保 Pandas 與 FastAPI 都能運作，啟動伺服器：`uvicorn w801_stock_screener:app --reload`。
  3. 前去 `http://127.0.0.1:8000/`，你一開始會看到所有 5 檔預設的股票一字排開。
  4. 現在我們來扮演嚴格的投資機構！在控制台填入：「最高價格 150，PE 最高限制 20」。
  5. 按下光速篩選按鍵。
  6. 觀察網址列是不是多了一條小尾巴 (`?max_price=...`)？
  7. 觀察下方的結果！是不是只剩下符合條件的那兩檔便宜的好公司（像是鴻海跟開發金）留下來？那些高價跟高本益比的科技股都被過濾掉了！
